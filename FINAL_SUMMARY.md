## 🎨 نظام الألوان

### Light Mode
```
🎯 أساسي: أخضر (#2D6A4F) + برتقالي (#F8961E)
📱 خلفية: تدرج أخضر فاتح ← → برتقالي فاتح
📝 نصوص: أخضر داكن (#1B4332)
```

### Dark Mode
```
🎯 أساسي: أخضر متوهج (#40C980) + أسود (#0F1117)
📱 خلفية: تدرج أسود عميق ← → أسود أعمق
📝 نصوص: أبيض ناعم (#E8EAEE)
```

## 🔄 كيفية الاستخدام

### 1. في أي Component
```typescript
import { useTheme } from './contexts/ThemeContext';

const MyComponent = () => {
  const { isDarkMode, toggleTheme } = useTheme();
  
  return (
    <div style={{
      backgroundColor: isDarkMode 
        ? 'var(--dm-surface)' 
        : 'white'
    }}>
      محتوى
    </div>
  );
};
```

### 2. الحفظ التلقائي
- يتم حفظ اختيار المستخدم تلقائياً
- عند العودة، يتم استرجاع نفس الـ Theme

### 3. احترام النظام
- إذا لم يختر المستخدم شيء، يتم احترام تفضيل النظام
