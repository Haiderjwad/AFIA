## ✨ التأثيرات والرسوم المتحركة

### 1. Fade Transition (على الخلفية)
```
Light: لون دافئ ────────────── (300-700ms) ────────────── Dark: لون بارد
```

### 2. Scale Animation (للـ Logo)
```
البداية: scale(0.8) opacity(0)
النهاية: scale(1) opacity(1)
المدة: 1000ms
```

### 3. Slide Animation (للنصوص)
```
البداية: translateY(24px) opacity(0)
النهاية: translateY(0) opacity(1)
المدة: مختلفة مع delay
```

### 4. Progress Bar Animation
```
الحركة: 0% → 100% على مدة 3 ثواني
اللون: ديناميكي حسب الـ Theme
التأثير: إضاءة (glow) متغيرة
```

## 🎯 خط سير المستخدم

```
1. الدخول للتطبيق
   ↓
2. عرض SplashScreen مع التفضيل المحفوظ
   ↓
3. (اختياري) المستخدم يضغط على زر التبديل
   ↓
4. تغيير الـ Theme بسلاسة
   ↓
5. الذهاب لصفحة Login بنفس الـ Theme
   ↓
6. (اختياري) المستخدم يمكنه التبديل مرة أخرى
   ↓
7. التفضيل يُحفظ تلقائياً
   ↓
8. عند الزيارة القادمة، يتم تطبيق نفس الـ Theme
```

## 🎬 المقاييس والأبعاد

### الشاشة
- **الحد الأدنى:** 320px (Mobile)
- **الحد الأقصى:** 2560px+ (Desktop)
- **نسبة العرض:** Responsive على جميع الأجهزة

### البطاقة (LoginView)
- **العرض:** max-w-md (28rem / 448px)
- **التقريب:** rounded-[2.5rem] (40px)
- **الارتفاع:** Auto (يعتمد على المحتوى)

### الأيقونات
- **الحجم الصغير:** 16-18px
- **الحجم المتوسط:** 20-24px
- **الحجم الكبير:** 40-56px

### النصوص
- **الحجم الأساسي:** 16px (body)
- **العناوين الكبرى:** clamp(2.8rem, 8vw, 4.5rem)
- **العناوين الصغرى:** 14px
- **النصوص الدقيقة:** 9-11px

## 🔄 الانتقالات الزمنية

```
300ms  ← لتغيير الألوان الأساسية
500ms  ← للـ Modals والحاويات الكبيرة
700ms  ← للخلفية والتأثيرات الكاملة
1000ms ← للـ Logo والرسوم الكبرى
```

## 💡 الملاحظات الهامة

✨ جميع الانتقالات سلسة وليست حادة
✨ الألوان لا تخالف معايير الوصولية
✨ النصوص دائماً قابلة للقراءة
✨ الأيقونات واضحة في الـ Modes
✨ الخلفيات لا تشوه النصوص
✨ التأثيرات لا تشتت الانتباه
